Event Check-In App

The Problem Statement

Design an application for Guest Checking-In during events for Event Managers.

The app should work in tandem with the other apps of EventForte ecosystem.

Interview Insights

User feeddback for the current application for Event Check-In segment.

The current app has different modes like diagramming (to set up virtual furnitures like table, stage), Virtual tour to client, Guest List, Seating Chart, Check In.

“Guests arrive in a short duration and are unwilling to stand in queue for longer period. Faster check-in would be helpful.”

“The app gives access to all the featuers to all the members. Junior members of the team in-charge of the checking-in guest have access to all modes which increases chances of mistakes.”

“Check In and Seating Chart are the only two features which are need during the event, rest of the features are used pre-event.”
